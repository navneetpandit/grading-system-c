# 🎓 Grading System in C

A simple program written in **C** that calculates a student's grade based on their marks.

---

## 🚀 Features
- Takes marks (0–100) as input
- Outputs grades:
  - A (90–100)
  - B (80–89)
  - C (70–79)
  - D (60–69)
  - Fail (< 60)
- Includes **input validation** for marks outside 0–100
